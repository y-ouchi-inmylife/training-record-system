# 用語集

システムで使用する用語の定義です。
設計書・コード内で用語を統一するために参照してください。

## ビジネス用語

| # | ビジネス用語 | 説明 |
|---|-------------|------|
| 1 | 内部ID | トレーニング機関内で使用する利用者の識別番号。DBの主キーとは別に、業務上で使う番号として保持する。新規登録時は現在の最大値+1で自動採番、編集画面で手動変更も可能 |
| 2 | 支援状態 | 利用者の現在の支援状況（例: 支援中、連絡待ち、支援終了）。ダッシュボード表示フラグでダッシュボードへの表示可否を制御可能 |
| 3 | フェーズ | 利用者の支援段階（例: 個人支援、集団支援、就職支援） |
| 4 | 所感 | トレーナーが記録する所見・気づき。トレーナー間で共有するが利用者には非開示 |
| 5 | 音声ファイル | 録音またはアップロードされた音声の実ファイル。音声記録の一種で、文字起こしの入力となる |
| 6 | 文字起こし | 音声ファイルをテキストに変換すること。Whisper APIを使用 |
| 7 | 要約 | 文字起こしテキストをAIで短くまとめたもの。Claude APIを使用 |

## 技術用語

| 用語 | 説明 |
|------|------|
| Whisper API | OpenAIが提供する音声文字起こしAPI。日本語を含む多言語に対応 |
| Claude API | Anthropicが提供する大規模言語モデルAPI。要約に使用（Haiku 4.5） |
| 非同期処理 | 処理の完了を待たずに次の操作に進める方式。文字起こし・要約の処理に使用し、ブラウザのタイムアウトを防ぐ |
| キュー（Job） | Laravelの非同期処理の仕組み。バックグラウンドでAPIの処理を実行する |
| MediaRecorder API | ブラウザ標準のAPIで、マイクからの音声をリアルタイムで録音する機能 |
| API（Application Programming Interface） | 外部システムとのデータ受け渡し口 |
