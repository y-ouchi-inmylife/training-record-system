# 手動テストチェックリスト

カウンセリング記録管理システムの手動テストチェックリストです。
各項目を実施し、チェックボックスにチェックを入れてください。

**最終更新日**: 2026-04-15

**実施者の権限レベル:**

| 権限 | 説明 |
|------|------|
| system_admin | システム管理者（設定画面のみアクセス可能） |
| admin | 管理者（すべての機能にアクセス可能） |
| staff | 一般（管理機能にアクセス不可） |
