# テスト仕様書（バッチテスト）: カウンセリング記録管理システム

## 1. 本書について

### 1-1. 目的

本書は、バッチ設計書（batch-design.md）に記載された全バッチについて、手動テストのテストケースを定義する。各バッチが仕様どおりに動作し、引数チェック・処理結果・ログ出力が正しく機能することを確認する。

### 1-2. テスト範囲

- コマンドライン（SSH接続）からの手動実行によるテストを対象とする。
- バッチ設計書の2章「バッチ詳細」に列挙された全バッチ（B-0101〜B-0302）を対象とする。
- 各バッチの処理結果は、DBの状態・ファイルの状態・ログ出力で確認する。
